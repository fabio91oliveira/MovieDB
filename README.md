# MovieDB App

## About the app

The app simply request MovieDB API to get a movie list

## Architecture

- Based on Android JetPack

## Basically using
* Kotlin
* Koin
* Androidx Refactor Library
* Lifecycle library (LiveData and ViewModels)
* Retrofit
* Glide
* Rx
